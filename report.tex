\begin{abstract}
This report documents the work done for UNSW's Robocup Standard Platform League (SPL) team \textbf{rUNSWift} during Semester 1, 2013.

The report describes 2013's Open Challenge entry, which applied Reinforcement-Learned sagittal and coronal stabilisation policies to the robots in various poses. The entry achieved third place overall in the Open Challenge.

The report also describes the new behaviours framework for the robots, known as NewSkillz. The framework makes writing robot behaviours a much simpler process by abstracting away much of the complexity and redundancy involved in state changes, and its consistent API allows for the easy implementation of a behaviours simulator. A port of 2012's behaviours is demonstrated to work on both the robots and in the simulator with no changes necessary between them.

\end{abstract}
